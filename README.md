# 🌌 Black Hole - Uma Jornada pelo Infinito

## React + TypeScript + Tailwind CSS

Uma experiência web única e marcante sobre buracos negros com animações incríveis.

### 🚀 Tecnologias Utilizadas

- **React 18** - Framework UI
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **GSAP** - Animações avançadas
- **Three.js** - Gráficos 3D
- **Lenis** - Smooth scrolling
- **Swiper** - Carrosséis responsivos
- **Framer Motion** - Animações de componentes
- **Shadcn/UI** - Componentes prontos

### 📦 Instalação

```bash
npm install
npm run dev
```

### 🎯 Recursos

- ✨ Visualização 3D do buraco negro
- 🎬 Animações suaves com GSAP
- 📱 Design responsivo
- 🖼️ Galeria interativa com Swiper
- 🌟 Smooth scroll com Lenis
- 📊 Fatos científicos interativos
- 🎨 Tema dark com gradientes
