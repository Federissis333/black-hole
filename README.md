# 🌌 Black Hole - Uma Jornada pelo Infinito

## Svelte + TypeScript + Tailwind CSS

Uma experiência web única e marcante sobre buracos negros com animações incríveis.

### 🚀 Tecnologias Utilizadas

- **Svelte 4** - Framework reativo
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **GSAP** - Animações avançadas
- **Three.js** - Gráficos 3D
- **Lenis** - Smooth scrolling
- **Swiper** - Carrosséis responsivos
- **Animate.css** - Efeitos visuais

### 📦 Instalação

```bash
npm install
npm run dev
```

### 🎯 Recursos

- ✨ Visualização 3D do buraco negro com Three.js
- 🎬 Animações fluidas com GSAP
- 📱 Design responsivo
- 🎠 Galeria interativa com Swiper
- 🌠 Smooth scroll com Lenis
- 📊 Fatos científicos interativos
- 🎨 Tema dark com gradientes espetaculares
- ⚡ Reatividade eficiente do Svelte
